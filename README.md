# edutech-learning-analysis

정보·컴퓨터 교과에서 최소성취수준 보장지도(최성보) 대상 학생을  
학기 중간 시점에 사전 예측하기 위한 EduTech 데이터 분석 및 AI 예측 프로젝트입니다.

본 프로젝트는 교사가 예방지도를 위한 대상 학생을 선별해야 하는  
현장 업무 부담을 데이터 기반으로 경감하는 것을 목표로 합니다.

---

# 1. 프로젝트 배경 (Background)

2022 개정 교육과정에서는 **최소성취수준 보장지도**가 도입되었습니다.

다음 조건에 해당하는 학생은 보충지도를 의무적으로 실시해야 합니다.

- 성취율 40% 미만
- 결석률 1/3 이상

문제는 다음과 같습니다.

- 결석률은 직관적으로 확인 가능
- 성취율은 학기 종료 후 확정되는 결과 지표
- 학기 중간에는 예측 기준 부재

특히 정보·컴퓨터 교과는

- 모의고사 없음
- 수행평가 비중 높음
- 과정 중심 평가 구조

로 인해 사전 선별이 어렵습니다.

---

# 2. 프로젝트 목표 (Problem Definition)

학기 중간 시점의 학습·평가·행동 데이터를 기반으로  
최소성취수준 미도달 위험군(최성보 대상)을 사전 예측합니다.

---

# 3. 데이터 구성 (Dataset)

> 실제 학생 데이터는 비식별 처리하며 GitHub에는 더미 데이터만 포함됩니다.

## 3.1 평가 데이터

- midterm_score : 중간고사 성적
- final_score : 기말고사 성적
- performance_score : 수행평가 성적

## 3.2 학습 행동 데이터

- assignment_count : 과제 제출 횟수
- participation_level : 수업 참여도
- question_count : 질문 횟수
- night_study : 야간자율학습 참여 여부

## 3.3 출결 및 생활 데이터

- absence_count : 결석 횟수
- behavior_score : 상벌점

---

# 4. 예측 타겟 정의

성취율 < 40% → 최성보 대상 (1)  
성취율 ≥ 40% → 일반 학생 (0)
